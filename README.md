# 🛒 E-Commerce — Spring Boot (Work In Progress 🚧)
⚠️ This project is currently in active development.

Some features may be incomplete or subject to change.

---

## 📌 Overview
This is a Spring Boot-based eCommerce backend project built to practice real-world backend development concepts including:

- REST API development  
- Authentication and authorization  
- Database integration  
- Scalable backend architecture  

## 🎯 Project Goal
The goal of this project is to independently design and build a complete eCommerce backend system using Spring Boot.
Previously, I contributed to Spring Boot projects in collaborative environments. This project is intended to demonstrate full ownership of the entire system design, implementation, and deployment workflows.

## 🧠 Learning Objectives
- Build a full eCommerce backend from scratch
- Improve system design and backend architecture skills
- Implement secure authentication and authorization
- Practice production-level development workflows
- Gain experience in building scalable APIs

---

## 💻 Tech Stack
- Java  
- Spring Boot  
- Spring Security  
- Spring Data JPA  
- MySQL / PostgreSQL  
- Maven  
- JWT Authentication

## ✅ Completed Features
- User model, controller, and service layers implemented  
- Product model, controller, and service layers implemented  
- Basic authentication system with signup and login functionality  
- Password security using basic authentication mechanisms  
- Layered architecture (Controller → Service → Model/Repository)
